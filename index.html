graphing-calculator-cpp

The project I have spent the most thought and effort into. A complete graphing engine implemented in C++ without any external libraries, just pure math.

graphing-calculator-cpp — OpenGL
C++17 OpenGL Implicit Curves Polar Plots Derivatives Quadtree Symbolic Math
4+
Plot Types
10^-12
Zoom Range
60fps
Real-time

It began as a simple function plotting program and evolved into something I can really be proud of. It supports plotting of implicit equations such as x²+y²=r², polar plots, symbolic derivatives, and has a quadtree-based rendering system for accurate rendering of curves, including near singularities and cusps. All the rendering pipeline elements were implemented from scratch without any external math libraries.
